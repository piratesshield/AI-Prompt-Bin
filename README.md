HOW TO INSTALL :

Open Chrome Extension Management Page ==> Go to chrome://extensions in your Chrome browser.

Enable "Developer mode" by toggling the switch in the top-right corner .

Load the Extension ==> Click on "Load unpacked" and select the folder containing your extension files.

Chrome will load the extension, and you’ll see it listed in your extensions .
